	\section{Introdução}
		\begin{flushleft}
		Com a preocupação com a segurança em todas as áreas, a identificação
correta e segura das credenciais de uma pessoa se torna de extrema importância.
Com a necessidade de vários documentos diferentes para as mais variadas
funções, surgem vários problemas, desde falsificação e roubo, até simples
desorganização.\\
	O intuito deste trabalho é criar uma infra-estrutura para emissão de passaportes
eletrônicos na plataforma Javacard, baseado no padrão ICAO 9303 \cite{ICAO}, tendo em
mente a segurança das informações. O sistema de emissão deverá prover um
aplicativo que colete os dados e produza um cartão funcional
Este cartão deve possuir todas as informações de identificação da pessoa,
sendo desnecessário outro documento de identificação promovendo a idéia de
cadastro único.\\
O projeto será efetuado juntamente com o LabSec e o professor responsável,
com a utilização do aplicativo JMRTD\cite{JMRTDSite}, que será adaptado para os fins
deste projeto, e projetos anteriores do LabSec que servirão de base para
desenvolvimento do produto final.
		\end{flushleft}
